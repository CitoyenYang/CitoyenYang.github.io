INFR08025 Computational Logic Lecture Notes (Introduction to Computation) Fall 2023

发表于 2023-11-28 更新于 2024-01-25 分类于 计算机科学
本文字数： 22k 阅读时长 ≈ 39 分钟

INFR08025 Introduction to Computation 的 Computational Logic 部分的笔记，只能说这门课非常的 Edinburgh，推荐文献都是 Mathematics Method in Linguistics，Computational Logic 这一部分像是 Mathematical Logic 与一小部分 Finite Automata 结合，其中代码实现部分使用的是著名的函数式编程语言 Haskell。
Haskell 这门语言也十分的 Edinburgh，Edinburgh 的 Prof. Philip Wadler 正是 Haskell 的主要设计者之一。虽然 Haskell 天天被车（连 bit 都定义不了，没有 loop 只能 recursion），但独特的函数式编程语法也委实能给不同编程水准的学生独特的程序设计思路，毕竟大名鼎鼎的 MIT6.001 Structure and Interpretation of Computer Programs 使用的都是函数式编程语言 Lisp 作为 EECS 的第一节公共必修课（虽然已经是老黄历了，现在也用 Python 了）
正如南京大学 PA 讲义中所说：语言的语法是次要的，对编程思维的锻炼才是最重要的。

聊聊《遗迹：灰烬重生》

发表于 2023-11-02 更新于 2024-01-24 分类于 电子游戏 ， TPS
本文字数： 268 阅读时长 ≈ 1 分钟

🐍GN 7/10 捡了芝麻丢了西瓜

作者为了所谓的魂味去强行缝合 TPS 与类魂，最终得到的结果是一个既有大部分射击游戏的缺点又有魂的缺点的作品，而魂本身的缺点在 TPS 这种游戏机制下也被放大了，射击游戏的堆怪与魂类的低容错相结合带来的体验可以说是比较差的。而作者对魂的优点的打磨，无论是关卡设计还是碎片化叙事，功力还是有所不足的，更别说射击游戏短平快的底子本身也与魂的这些优点有些许冲突，只能说是捡了芝麻丢了西瓜。
优点：

  • 优秀的枪械手感
  • 新鲜的流程设计

缺点：

  • 存在感极为薄弱的剧情与引导
  • 水平有限的碎片化叙事
  • 感知不强的成长系统

个人游戏时长：12.1 小时
